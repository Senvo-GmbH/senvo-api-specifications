Shipping

Up 
post /estimate-shipping-cost
Estimate Shipping Cost (estimateShippingCost)
Calculates projected shipping costs based on destination, weight, and optional dimensions

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json
  • application/xml

Request body

ShippingRequest ShippingRequest (required)
Body Parameter —

Return type

CostEstimate

Example data

Content-Type: application/json
{
  "charges" : [ {
    "amount" : 15.5,
    "description" : "Base shipping rate for package weight",
    "type" : "freight"
  }, {
    "amount" : 15.5,
    "description" : "Base shipping rate for package weight",
    "type" : "freight"
  } ],
  "estimatedDeliveryDays" : 3,
